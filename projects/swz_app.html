Smart Work Zones Application

Client MassDOT    Location Massachusetts    Duration 2016 - 2018   

KCUS is currently designing and developing a Smart Work Zones (SWZ) Application for the Massachusetts Department of Transportation (MassDOT). This application will enable the integrated centralized configuration, management, and monitoring of SWZ sites and field devices throughout Massachusetts. The SWZ application will also collect and fuse data retrieved from SWZs throughout the state. This data will be stored within the GoTime Data Warehouse for analytics and reporting purposes.
The application will also present this information in an intuitive manner through a web-based Graphical User Interface (GUI). The GUI will include a public-facing website that will allow for the controlled dissemination of work zone-related information. The SWZ application will also provide an XML feed that will provide real-time SWZ data to third party developers.
